## Main Logic

### Date Handling

```rpg
c                   time                    w_dato1
c                   subdur    1:*D          w_dato1
```
- **`time w_dato1`**: Retrieves the current system date/time (specifically the date part) into `w_dato1`.
- **`subdur 1:*D w_dato1`**: Subtracts 1 day from `w_dato1`. Thus, `w_dato1` ends up as yesterday's date.
